# Task 2 – Exploratory Data Analysis (EDA) & Business Intelligence

## ApexPlanet Software Pvt. Ltd. – Data Analytics Internship

## Project Overview

This project was completed as part of the ApexPlanet Data Analytics Internship (Task 2). The objective of this task was to perform Exploratory Data Analysis (EDA) on a retail sales dataset, write SQL queries to answer business questions, and build a static dashboard using Microsoft Excel.

The analysis was carried out using Microsoft Excel and SQL Server Management Studio (SSMS).

---

## Tools Used

- Microsoft Excel
- SQL Server Management Studio (SSMS)
- Microsoft Word
- GitHub

---

## Dataset

The dataset contains retail sales transaction data with customer details, product information, quantity sold, unit price, and total sales.

After data cleaning and feature engineering, the dataset contains:

- 1000 Records
- 15 Columns

---

## Exploratory Data Analysis (EDA)

The following analyses were performed using Microsoft Excel:

- Descriptive Statistics
- Gender-wise Sales Analysis
- Category-wise Sales Analysis
- City-wise Sales Analysis
- Product-wise Sales Analysis
- Histogram Analysis
- Scatter Plot
- Correlation Matrix
- Heatmap
- Static Dashboard

---

## SQL Business Questions

The following SQL queries were performed:

1. Total Records
2. Total Orders
3. Total Revenue
4. Orders by Category
5. Highest Revenue City
6. Average Order Value
7. Top 5 Products by Revenue
8. Average Age by Gender
9. Highest Average Sales by City
10. Total Quantity Sold
